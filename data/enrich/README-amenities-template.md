# Amenities enrichment template

## File

- **Template:** `gyms-no-amenities-template.csv` (generated by `npx tsx scripts/export-gyms-no-amenities-template.ts`)
- **After you enrich:** Save as `gyms-no-amenities-enriched.csv` in this folder and run `npx tsx scripts/apply-amenities-from-csv.ts` to apply to the database.

## How to use in LibreOffice Calc

1. **Open the template:** File → Open → `data/enrich/gyms-no-amenities-template.csv` (or paste the file contents into a new sheet).
2. **If opening the file:** When the import dialog appears, set encoding to **UTF-8**, separator to **Comma**, and tick “First row as header”.
3. **If pasting:** Paste into cell A1, then Data → Text to Columns → choose **Comma** as separator and use the first row as headers.
3. Fill the **amenities** column (column D). Use **comma-separated** amenity names, e.g.  
   `Showers, Parking, WiFi, Group Classes`
4. Save as CSV (e.g. `gyms-no-amenities-enriched.csv`) and keep the same columns: `slug,name,city,amenities`.
5. Do not remove or rename the `slug` column — the apply script uses it to match gyms.

## Allowed amenity names (use exactly)

Copy-paste these when filling the amenities column (you can combine any of them, comma-separated):

- 24/7 Access
- Air Condition
- Cafe
- Cardio Equipment
- Free Water
- Group Classes
- Hair Dryers
- Kids Friendly
- Locker Room
- Parking
- Sauna
- Showers
- Steam Room
- Swimming Pool
- Toilets
- WiFi
